What is Categorical Data?

A categorical variable is one for which the possible measured or assigned values consist of of a discrete set of categories, which may be ordered or unordered. Some typical examples are:

  • Gender, with categories Male, Female.
  • Marital status, with categories Never married, Married, Separated, Divorced, Widowed.
  • Side effects (in a pharmacological study), with categories None, Skin rash, Sleep disorder, Anxiety.
  • Political attitude, with categories Left, Center, Right.
  • Party preference (in Canada), with categories NDP, Liberal, Conservative, Green.
  • Treatment outcome, with categories no improvement, some improvement, or marked improvement.
  • Age, with categories 0-9, 10-19, 20-29, 30-39, etc
  • Number of children, with categories 0, 1, 2, etc..

Different Types of Categorical Variables

  • As the previous examples illustrate, categorical variables can have a varying number of different categories.

  • Different terminology to used to distinguish the number of categories.

- Binary variables (or dichotomous variables) is used to refer to a categorical variable with two levels.

- Polytomous variables is used to refer to a categorical variable with more than two levels.

Explanatory vs.Response Variables

  • The response variable is the dependent (Y) variable that we are interested in modeling.
  • The explanatory variable are the independent (Xs) or predictor variables help explain the response variable.
  • When the response variable is categorical, we cannot use our classic linear models like regression and ANOVA. Many of the assumptions of those models are not met when the response variable is categorical.

So, when the response variable is categorical, we use categorical data techniques to perform standard statistical analyses like hypothesis testing and modeling (two main focal points of this class).

Hypothesis testings main purpose is to determine if there is an association between two variables.

Model Building Approaches

Model-building does involve hypothesis testing.

In this class, we are going to focus on logistic regression (appropriate when the response variable is binary).

Logistic regression using a logit function.

In logistic regression, you are modeling a function of the probability of response, denoted as \(\pi\).

Logistic regression models the log odds.

\(logit(\pi)=\log(\frac{\pi}{1-\pi})\).

The logistic regression model is given by: \(logit(\pi)=\beta_0+\beta_1x_{1}+\beta_2x_{2}+...\)

There are hypothesis testing procedures testing if there is an association among the response variable and the explanatory variables (the Xs). The hypothesis tests invlove testing whether the \(\beta\) s are statistically different from 0.

A big advantage is that you can compute odds ratios (different from log odds!) that assist us in interpreting the model.
